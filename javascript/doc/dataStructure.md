## Object

Descriptor 属性描述符

```json
{
  "value": <String>,
  "writable": <Boolean>,
  "enumerable": <Boolean>,
  "configurable": <Boolean>
}
```

Descriptors 属性描述符集

```jsonc
{
  <String>:<Descriptor>, // key is propertyName
  ...
}
```

Entries 条目

```jsonc
[
  [key,value],...
]
```
